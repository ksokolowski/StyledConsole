Components:
===========

Core Classes:
- Console: Main API facade for terminal output
- FrameRenderer: Render frames with borders and styling
- BannerRenderer: ASCII art banner generation
- LayoutComposer: Stack, side-by-side, and grid layouts

Utilities:
- Color utilities (CSS4, gradients, interpolation)
- Text utilities (visual width, emoji handling, wrapping)
- Terminal detection (ANSI support, color depth)
- Validation (align, colors, dimensions)

Effects:
- gradient_frame() - Vertical gradients
- diagonal_gradient_frame() - Diagonal gradients
- rainbow_frame() - 7-color rainbow spectrum

Installation:
=============

pip install styledconsole

Quick Start:
============

from styledconsole import Console

console = Console()
console.frame("Hello, World!", title="Greeting")
console.banner("SUCCESS", font="slant")
