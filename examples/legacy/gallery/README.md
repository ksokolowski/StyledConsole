# Gallery Examples

Visual showcases of StyledConsole capabilities.

## Available Examples

### border_gallery.py

Displays all 8 border styles (SOLID, DOUBLE, ROUNDED, HEAVY, THICK, ASCII, MINIMAL, DOTS) with visual comparisons.

**Status:** ✅ Complete (M1)
